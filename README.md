# AWS Web Application Deployment

## Sobre o Projeto

Este projeto teve como objetivo realizar o deploy de uma aplicação web utilizando serviços da Amazon Web Services (AWS), simulando um cenário real de hospedagem em ambiente de nuvem.

Durante a execução do laboratório foram aplicados conceitos fundamentais de Cloud Computing, Infraestrutura, Linux, Redes e Administração de Servidores, permitindo disponibilizar uma aplicação web acessível pela internet através de uma instância Amazon EC2.

---

## Objetivos

* Provisionar uma instância Amazon EC2
* Configurar acesso seguro ao servidor
* Realizar conexão remota via SSH
* Instalar e configurar um servidor web Apache
* Integrar o servidor com um repositório GitHub
* Publicar uma aplicação web na nuvem
* Disponibilizar o site para acesso público

---

## Arquitetura da Solução

Cliente → Internet → Security Group → Amazon EC2 (Amazon Linux) → Apache HTTP Server → Aplicação Web

---

## Atividades Realizadas

✅ Criação de uma instância Amazon EC2

✅ Configuração de Security Groups para acesso seguro

✅ Conexão remota via SSH

✅ Instalação e configuração do Apache HTTP Server

✅ Instalação do Git no servidor

✅ Clonagem do projeto diretamente do GitHub

✅ Configuração do diretório web

✅ Publicação da aplicação web

✅ Testes de acesso e validação da aplicação

## Resultado Obtido

A aplicação web foi implantada com sucesso em uma instância Amazon EC2 utilizando o sistema operacional Amazon Linux.

Após a configuração do ambiente, instalação do Apache e publicação dos arquivos da aplicação, o site ficou acessível através do endereço público fornecido pela AWS, demonstrando na prática o processo completo de deploy em nuvem.
